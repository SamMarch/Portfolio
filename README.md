I am to create a portfolio that showcases all my new found skills using html and css. The code shows how some basic css can be appled along with active links that scroll the screen to where the link points to. It also showcases flex wrapping and media query for different screen sizes. In working on this portfolio I have learned many different techniques and lots of flex box issues that can occur.
