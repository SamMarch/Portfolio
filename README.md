# Sam March — Portfolio

Personal portfolio site showcasing my projects and skills as a software developer.

**Live site:** [sammarch.github.io/Portfolio](https://sammarch.github.io/Portfolio/)

## Tech Stack

- React 19
- TypeScript 5.9
- Vite 7
- CSS3 with CSS Variables

## Projects Featured

- **Affirming Minds** — Telehealth psychology practice website built with HTML, CSS, and JavaScript. [affirmingminds.com.au](https://affirmingminds.com.au/)
- **Portfolio Website** — This site, built with React and TypeScript.

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm

### Installation

```bash
git clone https://github.com/SamMarch/Portfolio.git
cd Portfolio
npm install
```

### Development

```bash
npm run dev
```

Opens at [http://localhost:5173](http://localhost:5173).

### Production Build

```bash
npm run build
```

## Deployment

Deployed to GitHub Pages via GitHub Actions. Pushes to `main` trigger an automatic build and deploy.
